# **Sistema de Gestão de Tarefas**

## ⚙️ **Requisitos Não Funcionais**

- Linguagem: **Java 17+**
- Framework: **Spring Boot**
- Segurança: **Spring Security + JWT**
- Banco de dados: **PostgreSQL**
- ORM: **Spring Data JPA**
- Front-end: **React.js + Axios + React Router**
- Comunicação: **API RESTful (JSON)**
